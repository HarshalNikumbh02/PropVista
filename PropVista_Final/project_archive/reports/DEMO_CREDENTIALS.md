# Demo Credentials

Use the following credentials to explore role-based actions on PropVista. All accounts share the same password.

**Default Password for all seeded users:** `Pass@12345`

### Accounts Matrix
| Username | Role | Purpose |
| :--- | :--- | :--- |
| `buyer` | Buyer | Browse listings, save to favorites, track inquiries and scheduled visits. |
| `seller` | Seller | Create/edit listings, upload cover and gallery photos, view inbound inquiries. |
| `admin` | Admin | Approve/Reject listing queue, manage users, and check system logs. |
| `superadmin` | Super Admin | Django Admin site access with full write permissions. |

*Note: You can use the quick-login demo links on the bottom of the landing page to switch between accounts instantly.*
