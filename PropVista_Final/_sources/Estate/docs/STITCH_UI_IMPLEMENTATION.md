# Stitch UI/UX Implementation Notes

The UI follows the uploaded Stitch style:

- Premium real estate landing page
- 3D-style property cards
- Glassmorphism navigation and dashboard panels
- Buyer, seller, and super admin dashboards
- Property catalogue and property details page
- Image gallery with thumbnail switching
- Map placeholder section
- Report dashboard with PDF export button

Extra improvements added:

- Role-based redirect after login
- Seller property status tracking
- Buyer favorites
- Inquiry email notification through Django console backend
- Basic recommendation section by city, property type, and similar price
- Pure-Python PDF export without extra libraries
- Seed command for demo users and properties
