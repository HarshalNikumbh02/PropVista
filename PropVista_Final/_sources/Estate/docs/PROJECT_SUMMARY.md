# Project Summary

## Name
Real Estate Sales and Purchase Management System

## Scope
This project is limited to property selling and purchasing. It includes seller listings, buyer search, inquiry, admin approval, transaction record, and reports.

## Main Flow
Seller adds property -> Super Admin approves listing -> Buyer searches property -> Buyer sends inquiry -> Seller responds -> Super Admin records sale/purchase transaction -> Property status becomes Sold -> Reports can be exported.

## UI/UX additions
- 3D property cards
- Glassmorphism navigation and dashboards
- Better property image gallery
- Google Maps placeholder section
- Inquiry email notification through console backend
- Property recommendations by city, price, and property type
- PDF export for reports
- Mobile-friendly Bootstrap layout
